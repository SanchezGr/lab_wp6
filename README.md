# ⚛️ Вебзастосування React.js

Вебзастосування створено в межах лабораторної роботи №6 з теми “Розробка вебзастосунків React.js”.

У проєкті реалізовано два завдання:
1. Просте вебзастосування React.js “Hello, World!”.
2. Вебзастосування для відображення рецепту “Картопляного пюре”.


## 🚀 Технології

Проєкт створено з використанням таких технологій:

- React.js — створення інтерфейсу користувача;
- JavaScript — логіка роботи компонентів;
- HTML — структура сторінки;
- CSS — стильове оформлення;
- Node.js та npm — створення й запуск React-проєкту;
- Git та GitHub — керування версіями та розміщення проєкту.

## 📌 Функціональні можливості

У вебзастосуванні реалізовано:

- відображення повідомлення “Hello, World!”;
- створення окремих React-компонентів;
- використання динамічних даних;
- відображення рецепту “Картопляного пюре”;
- стильове оформлення сторінки.

## 🛠️ Локальний запуск

Для запуску проєкту локально потрібно виконати команди:

```bash
git clone https://github.com/SanchezGr/lab_wp6.git
cd lab_wp6
